# GreenClassify
Vegetable Image Classifier using Flask and TensorFlow

## Description
Vegetable classification web app using TensorFlow and Flask.

## Project Files
- Code files (`app.py`, `templates/`, `static/`, etc.)
- Screenshots in `screenshots/`
- Project documentation in `docs/`
- Requirements file (`requirements.txt`)
